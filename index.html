Geolocation Data

Geolocation data can be collected automatically using a mobile device like a smartphone. Both GPS and cell tower plus wifi triangulation can be used to compute a person's coordinates.

Format

latitude,longitude,timestamp

time zone? polling interval? accuracy?

Producers

Android

GPS Logger on Android Google Location History Saga allows manual export to JSON that takes up to 24 hours Geotracks automatically uploaded to Geoloqi, which can be downloaded via API

Consumers

Interactive map visualization Route recommendations Dashboard or destinations, modes of transportation, distance traveled Daily schedule of location
